สรุปประสบการณ์ ความรู้สึก และสิ่งที่ได้เรียนรู้จากรายวิชา

ไฮไลต์ที่ภูมิใจ

การเรียนเชิงปฏิบัติ

ทำโปรเจกต์จริงตั้งแต่คิดโครง สร้างหน้าเว็บ จนปรับให้รองรับมือถือ — ได้เรียนรู้ทั้งกระบวนการและวิธีแก้ปัญหาเชิงปฏิบัติ

ค้นพบความชอบและเป้าหมายอาชีพ

การทำโปรเจกต์นี้ช่วยให้รู้ว่าชอบงานออกแบบหน้าเว็บและการปรับปรุงประสบการณ์ผู้ใช้ (UX) มากขึ้น และชัดเจนว่าต้องการทำงานด้านต่อไปนี้ในอนาคต:

ระเบียบวินัย

ในการเรียนและทำงานได้ฝึกการมึระเบียบวินัยในการเรียน เช่นการแต่งกายเข้าเรียน การส่งงานให้ทันตามกำหนด

สิ่งที่ได้เรียนรู้

  • ได้เรียนรู้พื้นฐาน HTML อย่างเป็นระบบ — การใช้ semantic tags (header, nav, main, section, article, footer) เพื่อให้โครงสร้างหน้าเข้าใจง่ายทั้งสำหรับคนและเครื่องมือค้นหา
  • เข้าใจหลักการจัดหัวข้อ (heading hierarchy) และการใส่ alt ให้รูปภาพเพื่อเพิ่มความสามารถในการเข้าถึง (a11y)
  • ฝึกการเขียน CSS แบบโมดูล / ใช้ class แยกความรับผิดชอบ (separation of concerns) ทำให้โค้ดอ่านง่ายและนำกลับมาใช้ใหม่ได้
  • ได้ลงมือกับ layout หลัก ๆ เช่น Flexbox และ CSS Grid — รู้จักเลือกเครื่องมือให้เหมาะกับปัญหา (แนวนอน/แนวตั้ง, กริดแบบสองมิติ)
  • เรียนรู้การทำ Responsive Design (mobile-first) ด้วย media queries, relative units (%, rem, vw) และการจัดการ spacing ให้สอดคล้องทุกขนาดจอ
  • เข้าใจแนวทางจัดการ typography — การเลือกฟอนต์, ขนาดตัวอักษร, line-height และการใช้ระบบ scale เพื่อความสม่ำเสมอ

ข้อเสนอแนะรายวิชา

  • ผู้ช่วยสอน (TA) ที่สอนในแล็บสอนอย่างละเอียด ทำให้ทุกคนเข้าใจ และเป็นที่พึ่งได้เมื่อผู้เรียนมีคำถามหรือข้อสงสัย
