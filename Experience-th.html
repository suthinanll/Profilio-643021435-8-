ประสบการณ์และการสะท้อนความคิด

สรุปประสบการณ์ ความรู้สึก และสิ่งที่ได้เรียนรู้จากรายวิชา

ไฮไลต์ที่ภูมิใจ

การเรียนเชิงปฏิบัติ

ทำโปรเจกต์จริงตั้งแต่คิดโครง สร้างหน้าเว็บ จนปรับให้รองรับมือถือ — ได้เรียนรู้ทั้งกระบวนการและวิธีแก้ปัญหาเชิงปฏิบัติ

ค้นพบความชอบและเป้าหมายอาชีพ

การทำโปรเจกต์นี้ช่วยให้รู้ว่าชอบงานออกแบบหน้าเว็บและ UX มากขึ้น และชัดเจนว่าต้องการทำงานด้านนี้ต่อไป

ระเบียบวินัย

ฝึกการมีระเบียบวินัยในการเรียนและการทำงาน เช่น การทำงานให้ตรงเวลา และการจัดลำดับความสำคัญ

สิ่งที่ได้เรียนรู้

  • พื้นฐาน HTML แบบเป็นระบบ และการใช้ semantic tags เพื่อความชัดเจนของโครงสร้าง
  • การจัดลำดับหัวข้อ (heading hierarchy) และการใส่ alt ให้รูปภาพเพื่อการเข้าถึง (a11y)
  • การเขียน CSS แบบแยกหน้าที่ (modular) เพื่อความนำกลับมาใช้ใหม่ได้
  • การใช้งาน Flexbox และ CSS Grid เพื่อแก้ปัญหา layout ต่าง ๆ
  • ออกแบบแบบ responsive (mobile-first) และการจัดการ spacing
  • การเลือกและจัดการ typography ให้สม่ำเสมอ

ข้อเสนอแนะสำหรับรายวิชา

  • การมีผู้ช่วยสอน (TA) ที่ชัดเจนและสามารถช่วยตอบคำถามในแล็บจะช่วยให้การเรียนรู้ดีขึ้น

ความรู้สึก/ความในใจ

ขอบคุณอาจารย์และผู้ช่วยสอนที่ให้การสอนอย่างตั้งใจ

สรุปสิ่งที่ได้

ทักษะแบบ Hard Skills

  • HTML5 semantic, alt/heading/meta
  • CSS Grid/Flex และการออกแบบ responsive

ต่อยอดหลังจบวิชา

  • นำทักษะไปใช้สมัครงานและทำงานในสาย Front-end / UX‑UI / Web Development
  • พัฒนาและดูแลเว็บไซต์จริงด้าน performance และ accessibility
